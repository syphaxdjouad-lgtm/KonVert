## 1. ANALYSE DU MARCHÉ

### Taille du marché

Le marché des landing page builders est estimé à **820M USD en 2026**, avec un CAGR de 14.6% jusqu'en 2035. Ce n'est pas un marché de niche — c'est une infrastructure critique pour tout e-commerçant qui scale.

Plus important : **+4.5 millions de boutiques Shopify actives dans le monde**, dont ~300 000 en France/Belgique/Suisse. L'ICP KONVERT (17-38 ans, Shopify/WooCommerce, fait ses pages seul) représente ~60% de cette base soit **~1.8M de cibles dans la zone EU francophone élargie**.

2 000 abonnés = **0.11% de ce marché**. C'est réaliste avec une exécution correcte.

### Analyse concurrents

| Concurrent | Utilisateurs | Positionnement | Prix/mois | Faiblesse |
|---|---|---|---|---|
| **PageFly** | 200 000+ stores actifs, 4.9/5 (11 900 avis) | Builder visuel Shopify polyvalent | 0–99$ | Trop générique, courbe d'apprentissage, pas d'IA générative |
| **Shogun** | ~50 000 stores, $115M levés | Enterprise, A/B testing | 39–299$ | Cher, fait pour les grosses marques, trop complexe pour les solos |
| **GemPages** | ~30 000 stores | Prix attractif, nombreux templates | 0–59$ | Interface datée, peu de différenciation IA |
| **Replo** | ~5 000 stores | Figma-to-Shopify | 99$+ | Ciblé devs/designers, pas les e-commerçants solos |

### Positionnement différenciateur KONVERT

Là où TOUS les concurrents vendent un **builder** (outil), KONVERT vend un **résultat** (page qui convertit, en 30 secondes).

La différence est fondamentale dans le messaging :
- PageFly : "Construis ta page"
- KONVERT : "Ta page est prête"

Autres angles de différenciation :
1. **Hook "1 page gratuite sans compte"** — aucun concurrent ne propose ça. Zéro friction, résultat immédiat, preuve avant achat
2. **IA générative** — le copy, la structure et le design sont générés automatiquement depuis l'URL produit
3. **42 templates haute conversion** — contre une interface de construction vide
4. **Cible francophone/EU** — PageFly et GemPages sont pensés pour l'anglophone. KONVERT peut dominer la niche FR/EU
5. **Déclencheur émotionnel précis** — "ta pub a floppé à cause de ta landing page" → message qui touche au moment exact de la douleur

### Verdict marché

Le marché est occupé mais pas saturé sur la niche "IA + résultat immédiat + e-commerçant solo francophone". C'est là que KONVERT doit s'engouffrer en priorité.

## 3. FUNNEL COMPLET — DÉCOUVERTE AU PAIEMENT

### Vue d'ensemble du funnel

```
DÉCOUVERTE
    ↓
TikTok / Pub / Blog / Referral / Bouche-à-oreille
    ↓ (taux: 100%)

LANDING PAGE konvert.app
    ↓ (taux: 35-45% cliquent sur "Essai gratuit")

PAGE /essai — Génération 1 page gratuite sans compte
    ↓ (taux: 70-80% vont au bout de la génération)

PAGE /preview/[id] — Résultat + CTA paiement + Countdown 7j
    ↓ (taux: 20-30% saisissent leur email)

SÉQUENCE EMAIL 5 messages sur 7 jours
    ↓ (taux: 8-15% convert en payant)

PAIEMENT Stripe — Starter 39€ ou Pro 79€
    ↓

DASHBOARD — Usage + feature discovery
    ↓ (taux: 40-60% renouvellent à J+30)

RÉTENTION + REFERRAL
```

### Taux de conversion estimés par étape

| Étape | Taux | Benchmark industrie |
|---|---|---|
| Visite → clic CTA essai | 35% | 25-40% pour SaaS freemium |
| Clic → génération complétée | 75% | — |
| Génération → email capturé | 25% | — |
| Email capturé → abonné payant | 12% | 8-15% SaaS trial-to-paid |
| Payant → renouvellement M+1 | 55% | 45-65% SaaS SMB |

### Ce que ça donne concrètement

Pour obtenir 1 abonné payant, il faut :
- 1 000 visiteurs uniques → 350 essais → 260 générations → 65 emails → **8 payants**
- **CAC organique : ~3-8€ si trafic gratuit**
- **CAC paid : dépend du CPC, estimé 20-50€**

### Points de friction à éliminer en priorité

1. La migration SQL `public_previews` doit être appliquée — BLOQUANT
2. Stripe LIVE doit fonctionner — BLOQUANT
3. La séquence email automatisée (J+1, J+3, J+5, J+7) doit tourner — BLOQUANT
4. La page `/preview` doit être irréprochable — c'est le moment de vente

## 8. STRATÉGIE VIRALE / REFERRAL

L'objectif : chaque abonné en ramène 0.5 en moyenne. À 2 000 abonnés, ça veut dire 1 000 abonnés supplémentaires "offerts".

### Programme affilié (long terme)

- **Commission : 25% récurrent** sur la durée de vie de l'abonné amené
- Paiement mensuel automatique via Stripe Connect
- Dashboard affilié dans le compte (clics, conversions, revenus)
- Cible : formateurs, agences, YouTubeurs, freelances Shopify

Exemple de calcul pour un affilié :
> Amène 10 abonnés Pro (79€/mois) → 10 x 79€ x 25% = **197€/mois récurrent**, sans effort une fois la recommandation faite

### Referral peer-to-peer (court terme, viral)

**Mécanisme "1 mois offert pour 3 invités" :**
- Dans le dashboard, bannière : "Invite 3 e-commerçants → ton prochain mois est offert"
- Lien unique tracké par abonné
- Activation automatique dès que les 3 s'inscrivent (pas forcément payants — friction minimale)

**Mécanisme "Powered by KONVERT" (passif) :**
- Option dans le dashboard : ajouter un mini-badge discret "Page créée avec KONVERT" sur la landing page publiée
- Contrepartie : 10€ de crédit/mois tant que le badge est actif
- Résultat : chaque page publiée devient une pub gratuite vers konvert.app
- Target : si 500 abonnés activent le badge, c'est 500 landing pages e-com qui pointent vers KONVERT

### Deal fondateur (urgence)

Pour les 500 premiers abonnés :
- Accès à vie au plan actuel avec -30% de réduction, à condition de partager 1 post/story sur leur compte e-com
- Preuves sociales + word-of-mouth en échange d'une remise permanente
- Timing : lancer dès mois 1, fermer l'offre à J+30

### Boucle de viralité naturelle

```
E-commerçant génère une page → publie sur sa boutique
→ son client voit la landing page → badge KONVERT
→ clique → tombe sur konvert.app → essai gratuit
→ convaincu → abonné
```
