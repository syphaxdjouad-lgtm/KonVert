# Cold email Konvert — Pack complet

## Fichiers
1. **01-icp-leads.md** — Qui cibler + où trouver les leads
2. **02-variantes-email.md** — 3 variantes d'ouverture (A/B/C)
3. **03-followups.md** — Séquence 3 follow-ups
4. **04-playbook-envoi.md** — Règles pour pas cramer konvertpilot.com
5. **leads-template.csv** — Tracker leads à dupliquer dans Google Sheets

## Quick start (J0)
1. Vérifie SPF/DKIM/DMARC → cf playbook §Setup
2. Confirme adresse d'envoi (recommandé : `syphax@konvertpilot.com`)
3. Score `mail-tester.com` doit être ≥ 9/10
4. Importe `leads-template.csv` dans Google Sheets
5. Source 10 leads via Meta Ad Library (45 min)
6. Envoie 10 emails Variante A — espacés 3 min entre 9h-12h

## Métriques target semaine 1
- 100 emails envoyés
- 5 replies (5%)
- 1-2 trials lancés
- 0 spam report
